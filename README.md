# 2026 商汤奖学金获奖学生网站

直接双击 `index.html` 即可本地预览，也可将本目录完整上传到任意静态网站托管服务。

## 文件说明

- `index.html`：页面结构
- `styles.css`：响应式视觉样式
- `students.js`：30 位学生的结构化公开数据
- `app.js`：排序、搜索、筛选与详情弹窗
- `assets/brand/`：官方主视觉及延展素材
- `assets/students/`：按学生 ID 命名的证件照

网站无第三方运行依赖。更新学生资料时，请同时保持 `students.js` 中的 `id`、`photo` 与照片文件名一致。
